# v-blog-vue
v-blog的前端,用vue.js开发

## 前端技术栈
- Vue.js
- Element-ui(做后台)
- axios(请求数据)
- vue-router(路由)

## 后端技术栈
- SpringBoot
- MySql
- Mybatis-Plus
